# netdef_slim
A python wrapper for tf to ease creation of network definitions.

## Usage
* Install tensorflow
* Install [lmbspecialops](https://github.com/lmb-freiburg/lmbspecialops/tree/eccv18).
* Run `source netdef_slim/bashrc` to use this package.
